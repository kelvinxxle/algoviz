Complexity Analysis

Operation Time Space

Policy Overview

Least Recently Used (LRU) discards the least recently used items first. This algorithm requires keeping track of what was used when, which is expensive if one wants to make sure the algorithm always discards the least recently used item.

Optimization Strategy

  • check_circle Combine Hash Map for O(1) lookup.
  • check_circle Double-Link for O(1) node removal/reinsertion.
Pseudocode / lru_cache.js content_copy 
function get(key) {
  if (!map.has(key)) return -1;
  
  // Move to front (MRU)
  updatePriority(key);
  return map.get(key).val;
}

function put(key, value) {
  if (map.has(key)) {
    removeNode(map.get(key));
  } else if (size == capacity) {
    // Evict LRU from tail
    removeNode(tail.prev);
    map.delete(tail.prev.key);
  }
  
  addNodeToHead(key, value);
  map.set(key, head.next);
}
